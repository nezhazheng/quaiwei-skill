# quaiwei - 去AI味引号清理工具

一个 Claude Code Skill，用于清理 AI 生成的中文文本中多余的引号。

## 问题

AI 生成的中文文本有个显著特征：到处加引号。比如：

- 使用**"Python"**进行开发 → 使用 Python 进行开发
- 这个**"API"**接口很好用 → 这个 API 接口很好用
- 实现了**"自动化"**部署 → 实现了自动化部署
- 所谓的**"最佳实践"** → 所谓的最佳实践

这些引号让文本一眼就能看出是 AI 写的。quaiwei 就是干这个的——把多余的引号去掉。

## 处理范围

**会去掉的引号：**

1. 中文语境中英文单词前后的引号
2. 普通词语上不必要的强调性引号

**会保留的引号：**

1. 直接引语（他说"没问题"）
2. 反讽用法（这个"专家"连基础都不懂）
3. 代码中的字符串字面量
4. 书名、文章标题的引用

## 安装

将 `quaiwei/` 目录放到 `~/.claude/skills/` 下即可。

```
~/.claude/skills/
└── quaiwei/
    └── SKILL.md
```

## 使用

在 Claude Code 中对任意文件使用 `/quaiwei` 即可触发，或者直接说"帮我去掉多余的引号"。

## 名称由来

quaiwei = 去AI味。去掉 AI 生成文本中那些一眼假的引号，让文本读起来更自然。
