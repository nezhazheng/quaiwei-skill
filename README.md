# quaiwei - 去AI味文本清理工具

一个Claude Code Skill，用于清理AI生成的中文文本中多余的引号和空格。

## 问题

AI生成的中文文本有两个显著特征：到处加引号，以及在中英文之间加空格。比如：

- 使用**"Python"**进行开发 → 使用Python进行开发
- 这个**"API"**接口很好用 → 这个API接口很好用
- 实现了**"自动化"**部署 → 实现了自动化部署
- 建议采用 **React** 框架 → 建议采用React框架

这些痕迹让文本一眼就能看出是AI写的。quaiwei就是干这个的——把多余的引号和空格去掉。

## 处理范围

**会去掉的：**

1. 中文语境中英文单词前后的引号
2. 普通词语上不必要的强调性引号
3. 中文与英文之间的多余空格

**会保留的：**

1. 直接引语（他说"没问题"）
2. 反讽用法（这个"专家"连基础都不懂）
3. 代码中的字符串字面量
4. 书名、文章标题的引用
5. 英文单词之间的正常空格
6. 代码块中的空格

## 安装

将 `quaiwei/` 目录放到 `~/.claude/skills/` 下即可。

```
~/.claude/skills/
└── quaiwei/
    └── SKILL.md
```

## 使用

在 Claude Code 中对任意文件使用 `/quaiwei` 即可触发，或者直接说"帮我去掉多余的引号"。

## 名称由来

quaiwei = 去AI味。去掉AI生成文本中那些一眼假的引号和空格，让文本读起来更自然。
